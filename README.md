#### 图片脸部检测
- 软件运行环境: Ubuntu + Python3.6 + Opencv 3.4.4
- 实现功能
    - 读入人物图片。
    - 使用Opencv脸部特征分类器识别图片中人物脸和眼睛。
    - 输出图片到屏幕上，同时绘制识别到的脸与眼睛。
    - 将识别的脸部提取出来，存成jpg文件。
    - 屏幕打印识别到的脸与眼睛数量。
